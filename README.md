# Operating-systems
Operating systems || Projects
This repository contains projects I implemented for the Operating Systems course. These projects focus on important concepts such as multithreading and memory management.

Project 1: Multithreading Sudoku Solver

In this project, I implemented a Sudoku solution validator using multithreading. The program checks whether a Sudoku solution is correct by dividing the work into multiple threads.

Different approaches were used:

A single-threaded approach
A multi-threaded approach using multiple threads (for rows, columns, and subgrids)

This project demonstrates how multithreading can improve performance and efficiency by running tasks in parallel.

Project 2: Paging and Memory Management

This project focuses on memory management using the paging technique. The program simulates how the operating system manages memory by dividing it into pages and frames.

It includes:

Address translation from logical to physical address
Page tables
Simulation of memory allocation

This project helped in understanding how operating systems efficiently manage memory and reduce fragmentation.
